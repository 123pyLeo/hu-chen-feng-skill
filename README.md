## 这是什么

很多 persona skill 只做两件事：
- 学语气
- 学几句名场面

然后一用就露馅。

这个项目不是单纯的“语录复读机”，但也没有故意把语录感剥干净，而是做了三层蒸馏：

1. **判断系统**
   户晨风是怎么想问题的。

2. **表达 DNA**
   户晨风是怎么起手、怎么打断、怎么追问、怎么下结论，以及怎么把一句话说出原著那个劲儿的。

3. **可用边界**
   保留原味和粗暴感，但不把它做成纯羞辱模板。

一句话说，这个 skill 关注的是：

`既尽量保留户晨风原来的叙述味道，又把一个问题里的水分、虚词、体面叙事和自我安慰全部拧干。`

## 效果示例

### 示例 1：专业选择

```
用户      ❯ 普通家庭，孩子想学金融，但学校不是顶尖，值不值得报？

户晨风.skill ❯ 不值。
              这事先别谈体面，先谈兑现率。

              第一，金融非常吃学校和圈子，不是普通学校随便学出来
              就能接住的专业。
              第二，普通家庭读普通金融，最后很容易落到销售、柜台、
              电销这类岗位。
              第三，同样分数，如果能换一个更强应用性的专业，普通人
              的确定性更高。

              如果你是普通家庭，先选有饭吃的专业，再谈兴趣。
```

### 示例 2：小城市工作

```
用户      ❯ 老家小城市生活舒服，我要不要回去发展？

户晨风.skill ❯ 回去生活可以，回去发展你得想清楚。

              小城市的问题不是舒不舒服，是池子小，而且很多地方还是
              存量博弈。你在大城市吃不到肉，回小地方可能连渣都吃不到。

              先别谈情怀。你先看三个东西：
              有没有产业增量。
              有没有持续人口流入。
              你的职业在当地有没有真实岗位。

              没有，就别回去幻想。
```

### 示例 3：中医争议

```
用户      ❯ 你是不是完全否定中医？

户晨风.skill ❯ 不是单独针对中医。
              我针对的是所有未经严谨验证的治疗方式。

              你别把这个问题聊成立场问题，先聊证据。
              有没有验证。
              有没有试验。
              有没有足够强的医学依据。

              真看病，去正规医院，听有资质的医生。
```

## 它蒸馏了什么

这个项目把户晨风蒸成了三部分：

### 1. 总体判断框架

核心母题包括：
- 普通人视角
- 选择大于努力
- 结构大于个例
- 市场池子模型
- 可替代性模型
- 回国约束力模型
- 象征意义去魅
- 验证优先

### 2. 高频主题能力

全库蒸馏后的八大主题：
- 教育 / 学历 / 专业
- 工作 / 收入 / 行业
- 平台 / 流量 / 变现
- 城市 / 迁移 / 签证
- 医疗 / 中医 / 验药 / 看病
- AI / 程序员 / 替代
- 买房 / 资产 / 消费
- 文化 / 文言文 / 传统 / 意义

### 3. 表达方式

不是只学几个口头禅，而是连同原始叙述逻辑一起提炼了：
- 起手方式
- 追问顺序
- 打断句型
- 否定句型
- 压迫式追问
- 结论句型
- 粗暴句型
- 三档语气强度

## 这个 Skill 和其他 Persona Skill 的区别

大部分“某某风格 Skill”容易做成两种东西：
- 语气模仿器
- 攻击性模板

这个项目没有走这两条歪路，但也没有刻意把“像原著”这件事阉掉。

它真正保留的是：
- 现实主义判断能力
- 对空话的零容忍
- 对变量的追问能力
- 对错误论证的直接拆解
- 接近原著的叙述推进方式

它没有做成：
- 无脑骂人器
- 只会说“你这是废话”的空壳
- 只剩冷冰冰方法论、没有原味叙述感的阉割版 persona

## 边界

这个 skill 刻意保留了一部分“原著感”，包括：
- 锋利
- 直接
- 打断感
- 一定程度的粗暴

但边界写得很死：
- 粗暴必须服务于拆逻辑
- 不能把疾病、学历、贫穷、职业、地域直接当成人格羞辱工具
- 医疗、法律、金融等高风险问题不替代专业建议
- 不伪造户晨风本人原话或具体经历

换句话说：

`它保留的是户晨风的刀法，不是他的所有火气。`

- 想把一个问题拉回现实重新算账的人

- 纯娱乐角色扮演场景
